# img_to_code

## 環境変数

.env

```env
GOOGLE_API_KEY=your_api_key
```

## ライブラリのインストール

```zsh
pip install -r requirements.txt
```

## brew install (Mac)

```zsh
brew install poppler
```

## 初期化

- initial.pyを実行してください

## 仕様

- input_pdfにpdfを入れるとコードが抽出される
